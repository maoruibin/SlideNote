## 三、功能详情

### 3.1 网页摘录 ✨

**详细文档**：见 [CHANGELOG.md](./CHANGELOG.md)

**核心功能**：
- 右键保存网页选中内容
- 按日期自动聚合
- 保存来源链接
- 保存成功通知

**保存格式**：
```
23:33

选中文字内容...

来自：[页面标题](URL)
```
