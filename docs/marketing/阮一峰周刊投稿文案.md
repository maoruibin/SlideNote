## 项目简介（100字以内）

SlideNote 是一款 Chrome 侧边栏插件。它将便利贴固定在浏览器侧边，**不遮挡页面内容**，让你随时记录和查找碎片信息（密钥、服务器地址、常用命令等）。支持跨设备自动同步，无框架依赖，打包仅 25KB。

## 为什么做这个项目？

写代码时经常需要临时查一些命令或配置，查完切回编辑器就忘了；在公司电脑学了新东西，回家想用却找不到；各种云服务的密钥、API 文档链接散落各处，每次都要翻半天。

现有的笔记插件都悬浮在页面上，遮挡内容，打断工作流。直到发现 Chrome 的 Side Panel API，可以固定在侧边栏不干扰主页面，于是做了 SlideNote。

**它不是一个笔记本**。它更像一个云剪贴板，专门存那些需要"快速记、随时查"的碎片信息。

## 核心亮点

| 亮点 | 说明 |
|------|------|
| **侧边栏，不遮挡** | 固定在浏览器侧边，与主内容并排，不遮挡任何东西 |
| **秒开，即用** | 打包 25KB，加载 < 100ms，点开立即可用 |
| **自动同步** | 多设备无缝同步，公司的密钥回家就能查 |
| **实时搜索** | 输入即过滤，秒速找到 |
| **支持 Markdown** | 密钥、命令、代码有格式，更清晰 |

---

## 使用场景

- **开发者**：存 API Key、服务器地址、常用命令、数据库密码
- **多设备工作者**：公司 Mac + 家里 Mac + 笔记本，信息随身走
- **内容创作者**：账号凭证、文案模板、AI 提示词
- **日常碎片**：IP 地址、会议记录、待办事项
