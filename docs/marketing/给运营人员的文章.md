## SlideNote 是什么？

**一个住在浏览器侧边的便利贴。**

点击 Chrome 插件图标，侧边栏就会展开，**不遮挡页面内容**：

```
┌────────────┬────────────────┐
│            │                │
│ SlideNote  │   网页内容      │
│ 侧边栏     │   完全可见      │
│            │                │
└────────────┴────────────────┘
```

**它不是一个笔记本**，而是专门存那些"随手记、随时查"的碎片信息。

---

## 为什么适合运营？

### 1. 侧边栏，不遮挡内容

这是最大的卖点。你可以在左边浏览网页，右边随时查看笔记，互不干扰。

### 2. 打开即用

不用单独打开 App，浏览器里就有。
上班打开电脑，点击插件图标，所有信息都在。

### 3. 自动同步

公司存的内容，回家电脑也能看到。
换手机、换设备，信息随身走。

### 4. 搜索方便

记得个大概？搜一下就能找到。

### 5. 支持 Markdown

v0.0.3 新增了 Markdown 支持，让文案模板、活动信息有格式，更清晰。

## 核心亮点

| 亮点 | 说明 |
|------|------|
| **侧边栏，不遮挡** | 固定在浏览器侧边，与主内容并排，不遮挡任何东西 |
| **秒开，即用** | 打包 25KB，加载 < 100ms，点开立即可用 |
| **自动同步** | 多设备无缝同步，公司的密钥回家就能查 |
| **实时搜索** | 输入即过滤，秒速找到 |
| **支持 Markdown** | 密钥、命令、代码有格式，更清晰 |

## 如何使用？

### 方式一：Chrome 应用商店（推荐）

[Chrome Web Store](https://chromewebstore.google.com/detail/appaojacakbjbbellfehlgjophpdpjom)

搜索「SlideNote」或直接点击上方链接安装。

### 方式二：开发者安装

```bash
# 克隆代码
git clone https://github.com/maoruibin/SlideNote.git

# 安装依赖
cd SlideNote && npm install

# 构建
npm run build
```

然后打开 Chrome 扩展管理页面，开启「开发者模式」，加载 `dist` 文件夹。
