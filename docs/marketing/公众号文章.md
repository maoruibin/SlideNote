## 起因

你是不是也有这种经历：

在公司电脑上存了阿里云的 AccessKey，回家想登录时怎么也想不起来；
把团队的 API Key 存在微信里，换手机后找不到了；
把常用账号密码记在备忘录里，结果几台设备之间不同步。

我也一样。所以我做了一个简单的解决方案——**SlideNote**。

---

## 什么是 SlideNote？

**SlideNote** 是一个住在浏览器侧边的便利贴：

- 固定在侧边栏，**不遮挡页面内容**
- 点击图标即可展开，秒开即用
- 你存的密码、API Key、服务器地址，自动同步到所有设备
- 支持基础 Markdown，让密钥、命令、代码更清晰

![SlideNote 界面](https://gudong.s3.bitiful.net/images/slidenote-marquee-zh.png)

---

## 它不是什么？

首先声明：**它不是一个笔记本**。

我不会用它来写文章、做笔记、整理知识。那是 Notion、Obsidian 们干的事。

SlideNote 更像是一个**云剪贴板**，专门存那些：
- 需要快速记下来的
- 随时可能要查的
- 跨设备都要能访问的

**碎片信息**。

---

## 最大的不同：不遮挡内容

这是我做 SlideNote 的核心原因。

市面上大部分笔记插件，都是悬浮在页面上方的：

```
┌─────────────────────────────┐
│     笔记插件（遮挡页面）      │
├─────────────────────────────┤
│                             │
│      你正在浏览的网页         │
│      （被挡住了一部分）       │
│                             │
└─────────────────────────────┘
```

SlideNote 用的是 Chrome 的 Side Panel API，固定在侧边：

```
┌────────────┬────────────────┐
│            │                │
│ SlideNote  │   网页内容      │
│ 侧边栏     │   完全可见      │
│            │                │
│            │                │
└────────────┴────────────────┘
```

**不打断，不遮挡，不打扰。**

## 核心亮点

| 亮点 | 说明 |
|------|------|
| **侧边栏，不遮挡** | 固定在浏览器侧边，与主内容并排，不遮挡任何东西 |
| **秒开，即用** | 打包 25KB，加载 < 100ms，点开立即可用 |
| **自动同步** | 多设备无缝同步，公司的密钥回家就能查 |
| **实时搜索** | 输入即过滤，秒速找到 |
| **支持 Markdown** | 密钥、命令、代码有格式，更清晰 |

## 技术上的选择

**我甚至没有用 React/Vue。**

为什么呢？

```
React 体积: ~100KB+
SlideNote 体积: ~25KB
```

对于一个碎片信息存储工具来说，框架的收益远小于成本。

我用了：
- Vanilla JS（原生 JavaScript）
- Chrome Storage Sync API（自带云同步）
- CSS Variables（设计系统）
- marked.js（Markdown 解析，仅 7KB）

代码量很小，维护成本很低，十年后依然能跑。

## 如何使用

### 方式一：Chrome 应用商店（推荐）

[Chrome Web Store](https://chromewebstore.google.com/detail/appaojacakbjbbellfehlgjophpdpjom)

搜索「SlideNote」或直接点击上方链接安装。

### 方式二：开发者模式

```bash
git clone https://github.com/maoruibin/SlideNote.git
cd SlideNote
npm install
npm run build
```

然后在 Chrome 中加载 `dist` 目录。
