# SlideNote：住在浏览器侧边的便利贴

> **Simple notes, always by your side**
> **侧边笔记，常伴左右**
>
> 作者: 咕咚同学

---

## 缘起

你有没有过这种时刻：

写 vim 时突然忘了某个快捷键，切到搜索引擎查完，切回来又忘了刚才的上下文；
配七牛云/坚果云时，又要去找之前存的 WebDAV 地址和密钥；
在公司电脑学了个新命令，回家电脑想用却找不到了；
填表时又要去翻手机里的身份证号、银行卡号...

这些信息**不是笔记**，它们不值得打开 Notion/Obsidian 去郑重其事地记录。

它们更像是**「随时要用、用完即走」**的碎片信息。

---

## 我的尝试

我也试过一些「网页笔记」插件，但它们都有一个共同问题：

**悬浮在页面上，遮挡内容。**

你想复制左边的网页内容，必须先关掉笔记框；想查看笔记，又得打开它。来回切换，打断心流。

直到我发现 Chrome 114+ 的 **Side Panel API**。

---

## Side Panel 是什么？

简单说，就是浏览器侧边的一个固定面板：

```
┌─────────────────────────────────────────┐
│  ← 你的网页                              │
│                                         │
│  (可正常浏览、复制、粘贴)                 │
│                                         │
├──────────────────┬──────────────────────┤
│                  │  ← SlideNote        │
│                  │                     │
│   剩余 90%       │   快捷键            │
│                  │   密钥              │
│   你的工作区      │   命令              │
│                  │   待办              │
│                  │                     │
└──────────────────┴──────────────────────┘
```

**关键是：它不打断你的工作流。**

- 左边是你的网页，可以正常操作
- 右边是 SlideNote，随时查看
- 两者互不干扰，无缝复制粘贴

---

## 它不是什么？

**首先声明：它不是一个笔记本。**

我不会用它来写文章、做笔记、整理知识。那是 Notion、Obsidian 们干的事。

SlideNote 更像是一个**云剪贴板**，专门存那些碎片信息。

## 我在 SlideNote 里存什么？

### 1. 快捷键备忘

```
Vim 快捷键
Ctrl+U  删除到行首
Ctrl+E  光标到行尾
dd      删除当前行
yy      复制当前行
p       粘贴
```

### 2. 云存储密钥

```
七牛云
Bucket: xxx
AccessKey: xxx
SecretKey: xxx

坚果云 WebDAV
URL: https://dav.jianguoyun.com
Username: xxx
Password: xxx
```

### 3. 服务器信息

```
阿里云 ECS
公网IP: 47.xxx.xxx.xxx
用户名: root
密码: xxxxxx
```

### 4. 个人信息

```
身份证: 420xxx1990xxxxxxxx
银行卡: 6222xxxxxxxxxxxxxx
手机号: 138xxxxxxxx
```

### 5. 命令备忘

```bash
# Mac 显示隐藏文件
defaults write com.apple.finder AppleShowAllFiles -bool true
killall Finder

# 查看端口占用
lsof -i:8080
```

## 核心亮点

| 亮点 | 说明 |
|------|------|
| **侧边栏，不遮挡** | 固定在浏览器侧边，与主内容并排，不遮挡任何东西 |
| **秒开，即用** | 打包 25KB，加载 < 100ms，点开立即可用 |
| **自动同步** | 多设备无缝同步，公司的密钥回家就能查 |
| **实时搜索** | 输入即过滤，秒速找到 |
| **支持 Markdown** | 密钥、命令、代码有格式，更清晰 |

---

## 技术细节

- **无框架依赖**：Vanilla JS，打包后 25KB
- **Chrome Storage Sync**：自动跨设备同步
- **自动保存**：停止输入 1 秒后保存
- **冲突处理**：多设备同时编辑自动合并
- **Markdown 支持**：让密钥、命令、代码有格式（v0.0.3 新增）

## 安装

### Chrome 应用商店

[Chrome Web Store](https://chromewebstore.google.com/detail/appaojacakbjbbellfehlgjophpdpjom)

### 开发者模式

```bash
git clone https://github.com/maoruibin/SlideNote.git
cd SlideNote
npm install
npm run build
```

然后在 `chrome://extensions/` 开启开发者模式，加载 `dist` 目录。
