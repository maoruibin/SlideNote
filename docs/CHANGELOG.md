# SlideNote 变更日志

> **Slide notes, always by your side**
> **侧边笔记，常伴左右**

---

## [0.0.6] - 2025-01-19

### 新增
- **夜间模式**：自动跟随系统深色模式设置，纯 CSS 实现

### 修复
- **删除 Bug**：修复删除笔记后编辑器内容不更新的问题
- **暗色模式**：修复社交图标颜色不适配的问题

### 优化
- **Toolbar 重新设计**：搜索/返回无边框，新建按钮占满中间区域
- **图标统一**：所有按钮图标统一为 16px
- **布局平衡**：Toolbar 左右间距对称
- **意见反馈**： 侧边菜单底部增加意见反馈入口

---

## [0.0.5] - 2025-01-15

### 新增
- **双击进入编辑模式**：预览模式下双击内容区域即可进入编辑模式，交互更直观

### 优化
- **产品名称显示**：完整显示品牌「SlideNote 侧边笔记 常伴左右」
- **英文名称优化**：英文显示「SlideNote - Always by your side」
- **开发体验优化**：默认构建改为 dev 模式，发版本使用 `npm run build:prod`
- **交互提示**：预览区鼠标指针改为 pointer，提示可双击编辑

---

## [0.0.4] - 2025-01-15

### 优化
- **全局编辑模式**：编辑/预览模式改为全局设置，切换笔记不打断工作流
- **默认编辑模式**：打开侧边栏默认进入编辑状态，符合"便利贴"快速记录的定位
- **新建笔记排序**：新笔记出现在列表顶部，方便快速访问
- **自动聚焦标题**：新建笔记后光标自动定位到标题输入框

### 修复
- 修复新建笔记排序逻辑错误（之前新笔记会排到底部）

---

## [0.0.3] - 2025-01-15

### 新增
- **Markdown 支持**：预览/编辑模式切换，让密钥、命令、代码更清晰
- **富文本复制**：复制时保留格式，可直接粘贴到邮件、微信、文档
- **复制为 Markdown**：一键复制源码，方便迁移到其他应用
- **导航箭头**：上/下箭头快速切换笔记
- **语法帮助**：内置 Markdown 语法快速参考
- **每条笔记记忆模式**：每条笔记记住自己的查看模式（预览/编辑）

### 变更
- 更新产品定位：**不是笔记本，而是浏览器侧边的便利贴**
- 核心卖点强调：**侧边栏，不遮挡内容**

## 版本说明

版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：

- **主版本号 (MAJOR)**: 不兼容的 API 修改
- **次版本号 (MINOR)**: 向下兼容的功能性新增
- **修订号 (PATCH)**: 向下兼容的问题修正

---

## 变更类型

- `新增`: 新功能
- `变更`: 功能变更
- `废弃`: 即将移除的功能
- `移除`: 已移除的功能
- `修复`: 问题修复
- `优化`: 体验优化
- `安全`: 安全性修复
